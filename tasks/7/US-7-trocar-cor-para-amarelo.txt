US-7 — Trocar cor para amarelo
===============================

CONTEXTO
--------
A aplicação necessita atualizar o esquema de cores para melhorar a interface visual. Esta história implementa a mudança da cor principal para amarelo conforme solicitado.

HISTORIA
--------
Como usuário,
quero que a aplicação exiba cores amarelas,
para ter uma interface visualmente mais atraente.

CRITERIOS DE ACEITACAO
-----------------------
1. Cor principal alterada para amarelo em todos os componentes visuais
2. Alterações aplicadas de forma consistente em toda a interface
3. Mudança testada e validada visualmente

REGRAS TECNICAS
---------------
- Seguir convencoes do CLAUDE.md
- Sem over-engineering
- Utilizar variáveis de cor reutilizáveis

CRITERIOS DE TESTE (QA)
------------------------
E2E:
  - Validar que os componentes principais exibem a cor amarela
  - Verificar consistência da cor em diferentes resoluções de tela

FORA DO ESCOPO
--------------
- Alterações em componentes não visuais
- Refatoração arquitetural
- Performance optimization
