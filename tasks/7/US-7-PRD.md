# PRD – US-7: Trocar cor para amarelo

## Problema
O site precisa de uma atualização visual de branding para refletir uma nova identidade corporativa. Atualmente, o design utiliza cores que não estão alinhadas com a nova direção da marca.

## Objetivo
Aplicar amarelo como cor principal em todo o site, particularmente no título, para implementar a nova identidade visual de branding.

## Sucesso Esperado
O site apresenta amarelo como cor predominante, com o título visualmente destacado em amarelo. A mudança é consistente e visível em todas as páginas.

## Requisitos Funcionais
1. Aplicar cor amarela como cor primária em todo o site
2. Garantir que o título do site seja exibido em amarelo
3. Manter a hierarquia visual e legibilidade do conteúdo

## Requisitos Não-Funcionais
- Acessibilidade: garantir contraste de cores adequado para legibilidade (WCAG)
- Consistência visual em todos os navegadores modernos

## Fluxo do Usuário
1. Usuário acessa o site
2. Vê a cor amarela aplicada como tema principal
3. Identifica o título em amarelo como elemento destacado
4. Navega pelo site mantendo a consistência visual da nova cor

## Critérios de Aceitação
- [ ] Cor amarela aplicada aos elementos principais do site
- [ ] Título do site exibido em amarelo
- [ ] Cores aplicadas de forma consistente em todas as páginas
- [ ] Contraste de cores adequado para legibilidade (não prejudica acessibilidade)
- [ ] Visual validado em navegadores modernos (Chrome, Firefox, Safari, Edge)

## Escopo Técnico
- Backend necessário: não
- Frontend necessário: sim

## Suposições
- Amarelo é a cor primária da nova identidade de branding
- A cor se aplica a elementos visuais principais, não necessariamente todo o background
- Legibilidade e acessibilidade devem ser mantidas
- Não há paleta de cores específica fornecida; será utilizado um amarelo padrão

## Fora do Escopo
- Alterações de layout ou estrutura do site
- Mudanças em componentes que não sejam visualmente relacionados à cor
- Redefinição de fontes ou tipografia
- Alterações funcionais ou de conteúdo
