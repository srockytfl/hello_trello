US-1 — Trocar cor para azul
======================

CONTEXTO
--------
Necessidade de ajustar a paleta de cores da aplicação para refletir nova identidade visual. A alteração da cor para azul melhora a coesão do design.

HISTORIA
--------
Como usuário,
quero que a cor da interface seja azul,
para ter uma experiência visual coerente com a identidade da marca.

CRITERIOS DE ACEITACAO
-----------------------
1. A cor azul deve estar aplicada em todos os elementos designados
2. A alteração deve ser consistente em todas as páginas/telas
3. O contraste de acessibilidade deve ser mantido (WCAG AA)
4. Não deve haver quebra de layout ou espaçamento

REGRAS TECNICAS
---------------
- Seguir convencoes do CLAUDE.md
- Sem over-engineering
- Utilizar variáveis de cores do sistema de design quando disponível

CRITERIOS DE TESTE (QA)
------------------------
API:
  - Validar se a configuração de cor é persistida e retornada corretamente

E2E:
  - Verificar se a cor azul é renderizada corretamente em todos os navegadores
  - Validar responsividade em diferentes tamanhos de tela
  - Confirmar acessibilidade de cores com ferramentas de contraste

FORA DO ESCOPO
--------------
- Modificar estrutura HTML ou adicionar novos componentes
- Alterar outras cores além do azul especificado
- Refatoração completa do sistema de temas
