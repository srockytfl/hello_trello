US-1 — Trocar cor para azul
=============================

CONTEXTO
--------
A aplicação apresenta cores que precisam ser atualizadas para refletir a nova identidade visual da marca. A mudança para azul é uma alteração importante na interface do usuário.

HISTORIA
--------
Como usuário,
quero que a interface exiba cores em azul,
para que a aplicação esteja de acordo com a nova identidade visual.

CRITERIOS DE ACEITACAO
-----------------------
1. Os elementos visuais principais devem usar a paleta de azul especificada
2. Todos os componentes afetados devem ter a cor azul aplicada consistentemente
3. A aplicação deve manter a acessibilidade (contraste mínimo WCAG AA)
4. As alterações devem ser visíveis em todos os estados (normal, hover, ativo, desabilitado)

REGRAS TECNICAS
---------------
- Seguir convencoes do CLAUDE.md
- Usar variáveis CSS/tema para manter consistência
- Sem duplicação de código de estilo
- Manter compatibilidade com navegadores suportados

CRITERIOS DE TESTE (QA)
------------------------
API:
  - N/A (alteração visual)

E2E:
  - Verificar que elementos com cor azul estão renderizados corretamente
  - Validar que o contraste entre texto e fundo atende padrões de acessibilidade
  - Confirmar que a cor persiste após navegação entre páginas

FORA DO ESCOPO
--------------
- Alterar paleta de cores além do azul especificado
- Modificar estrutura HTML dos componentes
- Implementar novo sistema de tema completo
