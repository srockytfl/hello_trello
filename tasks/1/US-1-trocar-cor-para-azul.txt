US-1 — Trocar Cor para Azul
======================

CONTEXTO
--------
O componente atual utiliza uma cor diferente que não se alinha com o design system.
Precisamos atualizar para azul para melhorar a consistência visual.

HISTORIA
--------
Como usuário,
quero ver o componente na cor azul,
para manter consistência com o design system.

CRITERIOS DE ACEITACAO
-----------------------
1. Componente primário deve exibir em azul
2. Cor deve seguir o padrão do design system (verificável visualmente)
3. Sem quebra de funcionalidade existente
4. Deve funcionar em todos os navegadores suportados

REGRAS TECNICAS
---------------
- Seguir convencoes do CLAUDE.md
- Usar variáveis de cor definidas no design system
- Sem inline styles — usar classes CSS ou variáveis
- Reusabilidade em mente para futuros componentes

CRITERIOS DE TESTE (QA)
------------------------
API:
  - N/A (alteração visual)

E2E:
  - Verificar renderização do componente em azul
  - Validar em diferentes resoluções de tela
  - Testar em navegadores Chrome, Firefox, Safari

FORA DO ESCOPO
--------------
- Refatoração completa do componente
- Mudanças em outros componentes relacionados
- Implementação de temas (dark mode, etc.)
